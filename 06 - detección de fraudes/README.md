# Detección de Fraudes

Proyecto de **Machine Learning**.
Analiza el dataset con 284k+ transacciones reales (anonimizadas), donde solo **0.17% son fraudes**.


## 🎯 Objetivos del Proyecto
- Exploración completa de datos (EDA)
- Manejo de clases desbalanceadas
- Modelos: **Regresión Logística** (supervisado) + **Isolation Forest** (no supervisado)
- Evaluación con métricas
- Comparación visual de rendimiento

## 📋 Flujo de trabajo

1. 📥 Carga datos
2. 📈 EDA
3. ⚙️ Preprocesamiento
4. 🤖 Modelado
5. 📈 Comparación ambos modelos
